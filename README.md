# Urdu Hierarchical News Dataset

## Description
This dataset contains multilingual (Urdu/English) news articles annotated with hierarchical labels (category and subcategory).

## Labels
- Level 1: Entertainment, Sports, Science & Technology
- Level 2: Domain-specific subcategories

## Format
CSV files with the following columns:
- id
- text
- category
- subcategory
- language

## Source
The data was collected from publicly available news websites.

## Ethics
All data is publicly available. No personal or sensitive information is included.
